# Virtual-Machine-Installation💻🛡️

## Objective
The objective of this virtual cybersecurity homelab is to develop and strengthen my hands on blueteam and SOC analyst skills by simulating real world enterprise environments. This lab will be used to practice system administration, system hardening, log collection, security monitoring, threat detection,pentesting, and incident investigation, in a safe, controlled, and legal, sandbox environment.
<br>

## Skills Learned

- **Operating system basics:** how OS installation works, partitions, system resource requirements, and user setup.
- **Virtualization concepts:** Built and managed a virtualized cybersecurity lab using VMware/VirtualBox/Hyper-V (Type 2 Hypervisors) based on multiple security learning objectives.
<br>

## Tools Used
- Microsoft Windows 11 (Primary O/S)
- Virtual Box (Type 2 Hypervisor)
- Kali Linux
